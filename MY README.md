# 01 HTML CSS Git: Code Refactor

Installation:
  1. Cloned Repo from Gitlab onto local computer into a new folder
  2. Create new repo in GitHub
  3. Make new folder on local computer
  4. Make a new index.html file inside this folder
  5. Add folder to new GitHub Repo
  6. Open file in Visual Studio Code
  7. Git push these folders to GitHub as you edit and make changes to code
  8. git commit will commit the changes to the GitHub Repo
  9. Deploy page through GitHub settings
